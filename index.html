About this Project

Hybrid Structure

This project is a series of pages with dynamic content that explains the relationship between the happiness and income in different states of the USA. On each page, we show US maps color coded, for different statistics. Users can hover over each state to obtain more details for better understanding/analyis.

Scene Utilization

There are five pages:

  • A welcome page that discusses about the project
  • A data vizualization of the Happiness rank of all the states in the USA
  • A data vizualization of the relationship between the Happiness and Medain Household Income for all the states in the USA
  • A data vizualization of understanding the relationship between Happiness and Adjusted Income (Income adjsted for the cost of living in that state) of all the states in the USA
  • A conclusion/references page.

Annotations

Annotation is represented by a tooltip which displays state name, happiness rank and score, median household income and adjusted income when the user hovers their mouse over a state on the map of the USA. As the user moves their mouse away from that state, the annotation associated with that state disappears and an annotation for the next state they hover over appears.

Parameters

Parameters in this project are: Reading the input .csv files as a function, loading JSON data to match the geo-location properties to the .csv data field "state", and passing color ranges to the domains to fill the maps.

Triggers

We provide buttons to navigate between pages. Every section is hyper linked to the next section as well the previouse section. The name of the target section is passed as an input paramter to the button and navgiation bar items.
